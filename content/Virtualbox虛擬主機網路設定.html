Virtualbox虛擬主機網路設定

Step1：前往雲端載點將檔案UB1804下載下來。

Step2：開啟Oracle VM VirtualBox Manager。

Step3：點擊右方視窗中的"NEW"按鈕來新增虛擬主機(名稱可隨意)，按下NEXT。

Step3：將Memory size調到4GB，按下NEXT。

Step4：將Hard disk選擇Use an existing virtual hard disk file(選項第三個)。

Step5：選擇檔案右方按鈕尋找"ub1804"檔案按下確定。

Step6：回到主畫面，點擊剛剛建立的虛擬主機ub1804，點擊右方視窗中的Setting前往設定。

Step7：左方選單中選擇Network，在右方視窗的Attached to選單中選擇Bridged Adapter(當使用者需要利用虛擬主機建立與 Host 位階相同的網路模式時，可以將虛擬主機的網卡設訂為 Bridged Adapter.)，確認後按下OK。

Step8：回到主畫面，啟動剛剛設定好的檔案，在右方視窗按下Start。

Step9：輸入帳號密碼(帳號：kmol2019密碼：kmol2019)

Step10：可以輸入 ifconfig 來查看IP。

Step11：輸入 cd /etc/netplan 按下Enter後，輸入1s -1按下Enter。

Step12：輸入 sudo vi 50-clould-init.yaml按下Enter。

Step13：輸入密碼kmol2019，前往設定。

(上下左右為 K J H L)

Step14：dhcp4:false
　　　　addresses:192.168.56.2/24
　　　　gateway4:192.168.56.1

Step15：編輯完成按下Ctrl+C退出，然後輸入sudo netplan apply同意設定。

Step16：輸入ifconfig 檢查ip。

Step17：輸入sudo survice cmsimfly restart重新啟動。
